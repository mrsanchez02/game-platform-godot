# Knights N' Slimes

A small 2D platformer developed with Godot as part of my game development learning journey.

Play as a knight, collect every coin, avoid slimes and hazards, and reach the end of the level.

## About the project

Knights N' Slimes started as a Godot platformer tutorial project.

After completing the original tutorial, I continued expanding it with custom
features such as a title screen, configurable audio, fullscreen support,
touch controls, controller support, haptic feedback and camera improvements.

## Features

- 2D platforming gameplay
- Coin collection objective
- Enemies and environmental hazards
- Keyboard and controller support
- Touch controls for mobile devices
- Music and SFX volume controls
- Fullscreen mode
- Controller vibration on jumps and player death
- Windows, Linux and Android builds

## Download and play

Playable builds are available from the repository's
[Releases](../../releases) section.

Available platforms:

- Windows
- Linux
- Android

> The current version is an early playable build and may contain bugs or
> incomplete features.

### Windows

1. Download the Windows ZIP file.
2. Extract its contents.
3. Run the executable.

### Linux

1. Download and extract the Linux build.
2. Grant execution permission if necessary.
3. Run the game executable.

### Android

1. Download the APK.
2. Allow installation from your browser or file manager.
3. Install and launch the game.

## Controls

| Action | Keyboard | Controller | Touch |
|---|---|---|---|
| Move left | A / Left Arrow | Left Stick / D-pad | Left button |
| Move right | D / Right Arrow | Left Stick / D-pad | Right button |
| Jump | Space | A / Cross | Jump button |

> Touch controls are displayed automatically on compatible devices.

## Development status

The game is currently in a playable alpha stage.

The existing level can be completed, but the project is still being used to
learn and experiment with Godot, game feel, user interfaces, input systems
and multi-platform exports.

Expect changes, unfinished areas and occasional bugs.

**Current version:** v1.0.2  
**Status:** Playable alpha  
**Engine:** Godot 4.7
